3. Final project:

   1. Goal:
      Build a program to manage a medical clinic. It should allow:
      - List patients
      - Insert, delete patients
      - Check patients:
        - Diagnose obesity from BMI (IMC)
        - Diagnose diabetes
        - Analize weekly calorie intake
      - Statistics: min, max, mean:
      - BMI (IMC)
      - Age
      - Weight
      - Heart Rate
